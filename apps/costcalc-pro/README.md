# CostCalc Pro - Version Plus sin limitaciones

## Que es CostCalc Pro

CostCalc Pro es una app para emprendedores y negocios gastronomicos que necesitan calcular con precision el costo por unidad de sus productos.

Se apoya en recetas, materias primas y gastos generales para obtener costos reales y mejores decisiones de precio.

## Caracteristicas principales

- Gestion de recetas personalizadas.
- Control de materias primas con costos y unidades.
- Registro de gastos generales.
- Calculo automatico del costo por unidad.
- Importacion y exportacion de datos.
- Generacion de reportes PDF.
- Multilenguaje.
- Funcionamiento 100% offline.

## Funcionalidades en detalle

### Recetas y costos
- Creacion/edicion detallada por ingrediente.
- Calculo automatico de costo unitario.

### Materias primas
- Registro y control de insumos con costos actualizados.

### Gastos generales
- Carga de costos indirectos para margen real.

### Reportes
- Import/export en formatos estandar.
- Generacion de PDF profesional.

## Instalacion (desarrollo)

```bash
git clone https://github.com/Kenshi1987/costcalc-pro.git
cd costcalc-pro
npm install
npm start
```
